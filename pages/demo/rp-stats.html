Résultats statistiques du recencement de la population en RDF

Introduction

L'Insee publie en RDF une sélection de résultats statistiques 2010 du recencement de la population : les estimations de population ventilées par commune (ou arrondissement municipal), age, sexe et type d'activité. Ces données correspondent au tableau détaillé "POP5", disponible aussi dans d'autres formats sur le site web principal de l'Insee (voir cette page, qui renvoie également sur la documentation détaillée associée aux données).

Modélisation des données

Les données sont structurées conformément au vocabulaire Data Cube, qui s'appuie sur le standard de diffusion de données statistiques SDMX. Conformément à ce standard, la structure des données est définie par une DSD (Data Structure Definition) qui décrit en particulier les dimensions du cube de données, le concept mesuré ainsi que divers attributs (unité de mesure, statut de l'observation, informations additionnelles, etc.).

Dans le cas présent, deux cubes de données sont définis. Le principal correspond au croisement entre commmune (référence au 1er janvier 2012), sexe, tranche d'âge et type d'activité. Pour le second cube, une dimension "arrondissement municipal" est substituée à la dimension "commune". Dans les deux cas, la mesure correspond à la population de 15 ans ou plus estimée au 1er janvier 2010.

Les listes de codes correspondant aux différents critères de répartition des observations sont exprimées sous la forme de structures de concepts SKOS.

Accès aux données

L'ensemble des données peut être téléchargé sous forme d'un fichier 7z (28 Mo) contenant deux fichiers au format Turtle : un pour le jeu de données sur les communes (pop5-2010-com.ttl) et un pour le jeu de données sur les arrondissements municipaux (pop5-2010-arm.ttl). Un second fichier contient la définition de la structure des données (pop5-dsd-2010.ttl) et les structures de concepts SKOS correspondant aux listes de codes pour les communes (cs-com-2012.ttl), les arrondissements municipaux (cs-arm-2012.ttl), le sexe (cs-sexe.ttl), les tranches d'âge (cs-ageq65.ttl) et le type d'activité (cs-tactr.ttl).

Nota bene : le format 7z est un format de compression ouvert supporté par de nombreux logiciels d'archivage, et en particulier par le logiciel open source 7-zip.
