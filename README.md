# Impala

Impala is a module of Insee's Statistical Metadata Repository (RMÃ©S). It exposes linked metadata on web pages and through a SPARQL editor based on [Yasgui](https://triply.cc/docs/yasgui-api).

## Getting Started

### Development

* Run Linting (oxlint)

```shell
pnpm lint
```

* Auto-fix linting issues

```shell
pnpm lint:fix
```

### Testing

* Run Unit Tests (Vitest)

```shell
pnpm test
```

* Run UI Tests (Playwright)

```shell
npx playwright test
```
